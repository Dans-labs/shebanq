Tools

This is a collection of tools based on the ETCBC database. Each tool is a notebook that reports on a research question. It contains the results, and provides a way to reproduce them. Notebooks contain code and documentation; any data they refer to, is stored on is available by following recipes on this website.

this website.

Verbal System in Poetry

Gino Kalkman: Verbal Forms in Biblical Hebrew Poetry: Poetical Freedom or Linguistic System? (Ph.D. thesis 2015). This tool contains a collection of notebooks in which Gino Kalkman presents his translations and analyses of the 150 Psalms in the Hebrew Bible and his theory on the use of the verbal forms in Biblical Hebrew poetry.

View index

Links to the individual notebooks for the psalms have been added to SHEBANQ as the note set verbsystem.

Verbal Valence

The context of selected verbs are categorized. For each of their occurrences it is recorded which kind of complements and other constituents are present. Depending on the pattern, sense of the verb is selected by means of a flowchart.

View notebook

Download notebook.

The outcomes have been added to SHEBANQ as the note set valence.

This tool is experimental. The algorithm and the results will change, and more verbs will be added over time.

If you have feed back on particular cases, pleas add your notes directly in SHEBANQ at the clause where it applies, and give it the label val_note.

Trees

This notebook composes syntax trees out of the ETCBC data. To this end, it imports the module etcbc.trees of LAF-Fabric, which contains the logic to synthesize trees from the data as it lies encoded in an EMDROS database and has been translated to LAF. etcbc.trees can also work for other data, such as the CALAP database of parts of the Syriac Peshitta. This notebook generates a tree for each sentence in the Hebrew Bible.

View notebook

Download notebook.

The results have been used for a Data Oriented Parsing experiment, which is reported in the Computational Linguistics in the Netherlands Journal 4 (2014) 105-120 (a copy is also archived at arxiv.org).

Phonetic Transcription

This tool contains the notebook that computes a phonetic transcription for the Hebrew text. The transcription is computed by applying patterns on the one hand, and by using the corpus for solving ambiguities that are not covered by the rules. There will be mistakes, but you may help to correct them, by giving feed-back in the form of corrections that you can write straight into SHEBANQ as manual notes to the sentence where you see offending words. Just use Notes view. If you like to program you own transcription, you can use an extensive set of identified hard cases, that are the by-product of this work.

SHEBANQ (itself)

SHEBANQ is powered by various databases, that are derived from the ETCBC database. Here is a collections of notebooks that carry out the conversion from the LAF representation of the ETCBC database up to the SQL files that can be imported in a MYSQL database.

This is not stuff for the normal user, but for the ones that need to recreate and/or maintain SHEBANQ, or for archivists that need to know the provenance of the data that powers SHEBANQ.

The workflow consists of executing an update script in python, which in turn calls a bunch of notebooks:

laf2shebanq (download)
We construct a relational database, passage, meant to support browsing of texts and highlighting of words. It contains the texts themselves, verse by verse, and it contains book and chapter information. The passage database also contains a lexicon, which is linked to the word occurrences.
lexicon (download)
This notebook prepares additional lexical data for LAF-Fabric: the lexicon and the pre-computed phonetic transcription. This data is added as an extra annotation package to the main ETCBC resource.
ketiv_qere (download)
This notebook prepares ketiv/qere information for shebanq. This data is added as a set of annotations inside the lexicon annotation package to the main ETCBC resource.
para_from_px (download)
This notebook prepares additional syntactical data for LAF-Fabric: the paragraph structure. This data is added as an extra annotation package to the main ETCBC resource.
mql (download)
This notebook tests the performance of SHEBANQ for taxing queries, precisely the kind that beginners are likely to put in SHEBANQ. See query {{=A(T("Too-many"), _href=URL('hebrew', 'query', vars=dict(id=1119, page=1)))}}.
plain (download)
This notebook reconstructs the plain text of the Hebrew Bible in various formats. You can also use it to render individual verses for quick copy and paste into your articles.
font-rendering (download)
This notebook grapples with the font rendering for several Hebrew fonts, in several browsers and in various contexts.
VerblessMothers (download)
This is an example of a check on the ETCBC encoding itself. Somebody discovered an instance of an object clause dependent on a verbless main clause. That is strange, and probably a coding error. This notebook is written to collect all instances of this error. (There are 52 instances in version 4b of the etcbc data.
notes (download)
This notebook defines the datamodel for notes in SHEBANQ.
versions (download)
This notebook defines a new datamodel for queries in SHEBANQ, and it converts data according to the old model to data according to the new model. The old model had no provisions for working with queries on multiple versions of the data. The new model remedies that.
loganalysis (download)
We read the apache2 log files, filter out irrelevant events, analyse remaining events, and produce usable statistics.
