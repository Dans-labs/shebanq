{{=T('About')}}

The shebanq service is the first data service of ancient-data.org.
Shebanq is a search engine for the Hebrew Bible, powered by the ETCBC4 linguistic database, formerly know as WIVU.
The data is archived for open access, and the program code is open source. A list of sources needed to recreate all underlying data processing is included below.

{{=T('License and Re-use')}}

The creators of the linguistic annotations that drive SHEBANQ, the ETCBC, want to facilitate optimal access to their data by researchers. All operations that are needed for the scholarly process are allowed. You are allowed to download the data and the code, build your own workflows on top of the data and even to redistribute your versions of the data and the code. The licence is:
This work is licensed under a Creative Commons Attribution-NonCommercial 4.0 International License. That means:

  • You may not use the data for commercial applications;
  • You must give proper attribution to the data when you use it in new applications, preferably by citing its persistent identifier urn:nbn:nl:ui:13-048i-71

The ETCBC prefers to stress the potential of using its data, rather than the legal restrictions in doing so. It has published data, tools and documentation online that enables teams of biblical scholars and developers to reproduce all data processing that is needed for SHEBANQ. The recommended tool to start with is {{=A('laf-fabric', _href='https://github.com/ETCBC/laf-fabric', _target='_blank')}}. Its documentation describes in detail how to download the data and set up data-processing workflows.

The ETCBC welcomes contributions that improve and enrich the interface of SHEBANQ. We are also open to implement mutual deep linking with other biblical data sources.

We have use the Emdros export version of the data, converted it to LAF, and archived it at DANS, a national research data archive in the Netherlands. Its repository is called EASY, and there, among many other data sets you find the Hebrew Text Database of the ETCBC.

ETCBC database version 4b

Snapshot taken from the ETCBC's jacob-server on 2015-05-04. This version is not archived, and not finished.

We will refresh this version regularly, until the ETCBC provides a stable, finished version, which we will call 4s.

In the meanwhile, the data that corresponds to the currently loaded 4b version can be downloaded from my SURFdrive. This data has been compiled by LAF-Fabric, and it can be used directly by your local LAF-Fabric installation.

ETCBC
The ETCBC organization on Github has several repositories related to its Hebrew Text Database.
shebanq-doc
Feature-by-feature documentation of the Hebrew Text Database. Here you find the meaning of the linguistic annotations to the text.
LAF-Fabric
The Hebrew Text Database is archived in the LAF format. LAF-Fabric is a data preprocessing tool that can handle LAF. It is a Python package. On top of it there are packages specialized in the peculiarities of the ETCBC data.
LAF-Fabric notebooks
This a repository full of executable notebooks, showing a wide range of things that can be accomplished. A recommended way to do data processing with the ETCBC data is using the concept of the IPython Notebook. In a notebook you can describe, document and execute Python code. Such notebooks can easily be shared through github and NBviewer.
DANS-Labs
The DANS-labs organization on Github contains the SHEBANQ project.
SHEBANQ
This is the code that drives the SHEBANQ website. It is based on web2py, whose code is also on Github.
EMDROS
The engine that performs MQL queries against an Emdros text database. Written in C++, by Ulrik Sandborg-Petersen.
