Methods

This is a collection of methods used in constructing the ETCBC database. Each method is a notebook that tackles a specific encoding or modeling problem. It contains the results of applying the method, and provides a way to reproduce them. Notebooks contain code and documentation; any data they refer to, is available by following recipes on this website.

Overview

The basis of the ETCBC method of encoding is the distinction between distributional concepts and functional concepts and how they hang together.

The assignment of values to features of the data is done through a sytem of signals. Multiple signals suggest certain values, and if they are in disagreement, a weighted vote is performed to reach a conclusion.

The ETCBC encoding is the result of many iterations of algorithmic work and evaluations by humans.

References

Here is a collection of diverse literature references about the ETCBC encoding.
