Welcome¶

Caution

Version etcbc4c is available from text-fabric-data. The set of features has changed slightly. A corresponding new version of this feature documentation is available at text-fabric-data: docs.

Data version etcbc4c is not (yet) available in SHEBANQ, only in Text-Fabric, the successor of LAF-Fabric. LAF-Fabric stays around in order to run legacy notebooks. It is recommended to use Text-Fabric for new work.

Welcome to the documentation of shebanq. The purpose of this documentation is to help you write MQL queries and LAF-Fabric scripts on the ETCBC4 database of the text of the Hebrew Bible. It is a reference manual that brings all details you need to know under your finger tips.

The MQL quick reference guide (pdf) is a good starting point. A deeper insight in how MQL is structured and what it can do for you is in Ulrik Petersen’s MQL Query Guide (pdf).

For an everyday reference to the ETCBC4 features, start with the Features (by category).

Much material is based on Constantijn Sikkel’s Description of Quest II Data File Format. However, these descriptions are not self-contained, and they have been written for a different representation of the data.

We are in the process of adding more in-depth reference material, not only about the text and its features, but also about the lexicon. More information about the lexicon preparation is in this notebook.
