Clause/Phrase(-atom) type typ¶

frequency table of values

The linguistic kind of objects.

This feature is present on objects of type clause(_atom), phrase(_atom).

Clause (atom)¶

For clause-like objects the values provide a classification on the basis of the constituents of the clause. This feature is also known as clause (atom) type.

Caution

I prefer a more informative definition.
