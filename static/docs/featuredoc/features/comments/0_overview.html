Features (by category)¶

Introduction¶

This is the key to the meaning of the features of the ETCBC4 database. There are several types of objects, which specify what features such objects have. Some database features are shared by all object types. We organize the object types as follows: sectional, word, and linguistic. See otype for a description of these types.

Caution

At the moment, not all features have been described clearly enough. See Features with cautions

- database generic features for all objects
- sections division in books, chapters, etc
- word all about the individual words
- syntax phrases, clauses, etc

Word features¶

– orthography several representations of the word occurrence
– lexical lexical information about the word
– morphology morphology classes of the word occurrence
– morphemes morphemes that are present in the word.
