KetivÂ¶

The unpointed representation of a ketiv word occurrence in Hebrew script when there is a divergent qere.

This feature is present on objects of type word.

When there is a ketiv-qere discrepancy, this feature contains the unvocalized ketiv. In those cases, but only in those cases, it is identical to the contents of g_word_utf8 feature.

See for an example Genesis 12:8

Caution

In cases where the ketiv is different from the qere, the qere is not present as a real feature that you can query in SHEBANQ. You can find ketivs in the g_word_utf8 - feature of words that have a masora sign. So query for:

[word g_word ~ '\*']

  • g_word_utf8.
